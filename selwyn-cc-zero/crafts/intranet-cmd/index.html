Intranet CMD

The new intranet page redefines the online visual style of CMD.

Working with the new visual identity of the CMD course designed by Grrr. I built a responsive intranet page showcasing the latest news, projects and course information. The new visual identity exists of a bold yellow color, big photos and flexible imagery. There is no logo instead the imagery illustrates the different aspects of the course.

Approach

Using HTML5 and CSS3 this simple and lightweight page is fully responsive, accessible and responsive. By using the picture element the browser only downloads the image at the needed size.
